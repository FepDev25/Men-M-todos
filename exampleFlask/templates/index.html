Metodos Numéricos

Este proyecto web está diseñado como una aplicación para la resolución y visualización de métodos numéricos utilizados en matemáticas e ingeniería. El objetivo es crear una interfaz didáctica para resolver ejercicios empleando los diferentes métodos vistos en la materia.

Indicaciones

En las siguientes secciones se presentan los métodos divididos por los diferentes temas que se han visto en la materia. Cada método tiene su propia interfaz, en donde se darán las respectivas indicaciones para realizar los ingresos correctamente.
