Método de Simpson 3/8

El método de Simpson 3/8 es una técnica de integración numérica que proporciona una aproximación a la integral definida de una función utilizando un polinomio cúbico para aproximar el área bajo la curva. Este método es útil para obtener resultados más precisos en comparación con el método de Simpson 1/3, especialmente cuando el número de subintervalos es mayor.

Indicaciones:

  • La función debe ser continua en el intervalo [a, b].
  • El número de subintervalos (n) debe ser múltiplo de 3 para aplicar el método de Simpson 3/8.
  • Las funciones deben ingresarse en términos de x. Ej: x + 45
  • Incluir los signos en cada una de las operaciones. Ej: 2*x + 3*x - (3*x/5) - 12
  • Para potencias se debe utilizar "**". Ej: x**2 - 13
  • Para funciones trigonométricas usar: sin(x), cos(x), tan(x)
  • Para constantes utilizar: "pi" para el número Pi, "E" para el número de Euler.
  • Para logaritmos utilizar: "ln(x)" para logaritmo natural. "log(x, base)" para logaritmos de diferente base. Ej: log(x, 10) + 9
  • Para indicar decimales se usa el punto. Ej: 3.7*x - 15

Ingresos:

  • Límite Inferior (a): Extremo inferior del intervalo de integración.
  • Límite Superior (b): Extremo superior del intervalo de integración.
  • Número de Subintervalos (n): Cantidad de subintervalos en que se divide el intervalo [a, b] (debe ser múltiplo de 3).
  • Función: La función continua a integrar.
