Métodos de Interpolación

La interpolación es un método para estimar valores intermedios entre puntos de datos conocidos. Existen varios métodos de interpolación, cada uno con sus ventajas y aplicaciones particulares.

Indicaciones:

Los valores de X y Y deben ingresarse como una lista de números separados por comas. Ej: 1, 2, 3, 4
Los nuevos valores de X deben ingresarse también como una lista de números separados por comas para obtener los valores interpolados.
Seleccione el método de interpolación deseado. Si selecciona Splines B-Spline, deberá ingresar el grado del spline (entre 1 y 5).

Ingresos:

  • Valores de X: Lista de valores de X conocidos.
  • Valores de Y: Lista de valores de Y correspondientes a los valores de X conocidos.
  • Nuevos valores de X: Lista de nuevos valores de X para los cuales se desea calcular la interpolación.
  • Método de Interpolación: Seleccione el método de interpolación (Lineal, Splines Cúbicos, Splines B-Spline, Polinomios de Lagrange).
  • Grado: Grado del spline (solo para Splines B-Spline).
