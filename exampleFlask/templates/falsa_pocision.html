Método de Falsa Posición

El método de falsa posición es un algoritmo numérico utilizado para encontrar raíces de una función continua.

Indicaciones:

Las funciones se deben ingresar en términos de x. Ej: x + 45
Se debe incluir los signos en cada una de las operaciones. Ej: 2*x + 3*x - (3*x/5) - 12
Para potencias se debe utilizar "**". Ej: x**2 - 13
Para funciones trigonométricas usar: sin(x), cos(x), tan(x)
Para constantes utilizar: "pi" para el número Pi. "E" para el número de Euler.
Para logaritmos utilizar: "ln(x)" para logaritmo natural. "log(x, base)" para logaritmos de diferente base. Ej: log(x, 10) + 9
Para indicar decimales se usa el punto. Ej: 3.7*x - 15

Ingresos:

  • Xi: Límite inferior del intervalo de búsqueda.
  • Xu: Límite superior del intervalo de búsqueda.
  • Función: La función continua de la cual se desea encontrar la raíz.
  • Máximas Pasadas: Número máximo de iteraciones permitidas para el cálculo.
  • Porcentaje Aproximación: Tolerancia para el error aproximado.
  • Porcentaje Verdadero: Tolerancia para el error verdadero.
